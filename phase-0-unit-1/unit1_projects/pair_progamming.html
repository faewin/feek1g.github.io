Defining pair programming:

Pair programming is a method for developing code between two programmers. It provides colloboration between two people and helps make better design of the webpage as well as enchancing a better learning experience. The programmer at the keyboard is usually called the "driver", and the other who is in charger of the overall direction is called the"navigator".

Overall

Pair programming increases code quality, because you adopt a "programming out loud mindset" which reduces the risk of error. Additionally it helps transfer skills by observing more experienced team members or partners code.
