This week I was introduced to the "to_proc" shortcut. The best way to explain it is by actually showing you. Take the code below for instance:


In this example the array object [1,2,3] calls map, and says "Give me a block of code, and I will execute it on every element in this array." It will iterate through each item in array [1,2,3] and will execute the block. In this case the the &, and symbol :to_s are being passed into map which invokes the to_proc method. Here we have the method to_proc being called on the symbol :to_s. This method returns a Proc object, which is just a block of Ruby code that performs specific operations. For the example above, it returns the result of calling to_s on each element of the array.
