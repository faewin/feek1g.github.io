What is SQL injection?

SQL injection is a code injection technique, that allows a hacker to gain access to a backend database. It is made possible as a result of an improperly coded website that allow SQL statements to be inserted into input fields, querying the database itself. In short it allows a hacker to effectively communicate directly with the database itself.

For example, through SQL injection a hacker can exploit the security vulnerability of the application, and write an SQL query that will show all the username and passwords in the database. This query will then be pasted into an input field such as a login field in a webpage, and once executed the hacker will have access to what should of been secured information or worse like the comic above delete the database

What can we do to prevent it?

Like the picture above, SQL injections can be prevented by sanitizing the database inputs.This means that the input is made invulnerable before being sent with the SQL query to the database. Before the query is sent to the database, it should be validated in some way.
